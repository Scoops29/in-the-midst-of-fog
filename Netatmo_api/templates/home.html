Netatmo script with webhook

simple home page with example calls

this is the resource you are at now, the home page:

http://127.0.0.1:443/

this resource generates a link that can be used to login to netatmo and authorize this script to access the device, it only needs to be done once:

http://127.0.0.1:443/login

this resource generates an access token with your credentials and sets up the webhook to receive event data:

http://127.0.0.1:443/setup

this resource is the webhook endpoint, it will not return anything as it is constantly listening for events, but it can still be accessed:

http://127.0.0.1:443/webhook

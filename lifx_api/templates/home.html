Lifx extended API

simple home page with example calls

this is the resource you are at now, the home page:

http://127.0.0.1:5000/

this resource returns the state of the device:

http://127.0.0.1:5000/api/devices/lifx

this resource switches the device on and off:

http://127.0.0.1:5000/api/devices/lifx/switch

this resource changes the colour, you need to add key value pairs at the end so the server knows what colour to change to, like so: "?colour=red":

http://127.0.0.1:5000/api/devices/lifx/colour?colour=red

this resource turns on the light for a certain time, you need to add key value pairs at the end so the server knows how long for, like so: "?time=3":

http://127.0.0.1:5000/api/devices/lifx/timer?time=3

this resource changes the colour for a certain time, you need to add key value pairs at the end so the server knows what colour to change to and for how long, like so: "?time=10&colour=red":

http://127.0.0.1:5000/api/devices/lifx/colourTime?time=10&colour=red

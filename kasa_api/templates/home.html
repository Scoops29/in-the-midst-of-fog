Kasa extended API

simple home page with example calls

this is the resource you are at now, the home page:

http://127.0.0.1:80/

this resource returns the state of the device:

http://127.0.0.1:80/api/devices/kasa

this resource switches the device on and off:

http://127.0.0.1:80/api/devices/kasa/switch

this resource turns on the switch for a certain time, you need to add key value pairs at the end so the server knows how long for, like so: "?time=10"

http://127.0.0.1:80/api/devices/kasa/timer?time=10
